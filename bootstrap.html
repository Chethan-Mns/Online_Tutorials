Bootstrap Basics

Welcome to the Bootstrap Basics tutorial! In this tutorial, you'll learn the basic concepts of Bootstrap, a popular CSS framework that simplifies web development.

Grid System

Bootstrap provides a responsive grid system. Here's an example of a two-column layout:

<div class="container">
    <div class="row">
        <div class="col-sm-6">
            <p>Column 1</p>
        </div>
        <div class="col-sm-6">
            <p>Column 2</p>
        </div>
    </div>
</div>

Components

Bootstrap includes pre-styled components like buttons, forms, and navigation bars. Here's an example of a Bootstrap button:

<button type="button" class="btn btn-primary">Primary Button</button>
