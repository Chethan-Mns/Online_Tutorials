HTML Basics

Welcome to the HTML Basics tutorial! In this tutorial, you'll learn the fundamental concepts of HTML, the markup language used to structure content on the web.

HTML Structure

HTML documents are composed of elements, each represented by tags. A basic HTML document structure looks like this:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Common HTML Tags

Here are some common HTML tags:

  • <h1> - <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (for links)
  • <img>: Image
  • <ul>, <ol>: Unordered and Ordered Lists
  • <li>: List Item

Feel free to explore more HTML tags as you continue your learning journey!
