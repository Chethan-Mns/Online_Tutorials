R Programming Basics

Welcome to the R Programming Basics tutorial! In this tutorial, you'll learn the fundamental concepts of R, a statistical programming language widely used for data analysis and visualization.

Hello World Example

Let's start with a simple "Hello, World!" example in R:

# Print Hello, World!
cat("Hello, World!\n")

Data Structures

R supports various data structures such as vectors, matrices, and data frames:

# Create a numeric vector
numeric_vector <- c(1, 2, 3, 4, 5)

# Create a character vector
character_vector <- c("apple", "orange", "banana")

# Create a matrix
matrix_example <- matrix(1:6, nrow = 2, ncol = 3)

# Create a data frame
data_frame_example <- data.frame(Name = c("John", "Jane", "Bob"),
                                  Age = c(25, 30, 22))
